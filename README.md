# Breakout3D

A 3D Breakout game implemented in C++ with OpenGL.

## Project Documentation
For a detailed technical overview, architecture, and feature list designed for developers and LLMs, please see [PROJECT_CONTEXT.md](./PROJECT_CONTEXT.md).

## Quick Start
1. Go to `Breakout3D/Breakout3D`
2. Run `make`
3. Run `./breakout3d` (or `make run`)
